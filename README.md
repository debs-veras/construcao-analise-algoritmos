# 🧠 Construção e Análise de Algoritmos

Este repositório contém implementações em C++ de diversos algoritmos clássicos Estudados na disciplina de **Construção e Análise de Algoritmos (CAA)**. O foco está no entendimento de diferentes paradigmas de projeto de algoritmos, bem como na análise de sua complexidade.

---

## 🚀 Conteúdo do Repositório

### 📂 Algoritmos de Ordenação e Diversos
Os seguintes algoritmos estão implementados diretamente na raiz do projeto:

- `CrivodeEratostene.cpp`: Algoritmo para encontrar números primos até um determinado limite.
- `Hanoi.cpp`: Solução clássica para o problema da Torre de Hanói (Recursão).
- `HeapSort.cpp`: Implementação do algoritmo de ordenação Heap Sort.
- `MergeSort.cpp`: Implementação do Merge Sort (Divisão e Conquista).
- `QuickSort.cpp`: Implementação do Quick Sort (Divisão e Conquista).
- `Ordenacao.cpp`: Outros algoritmos ou comparações de ordenação.
- `MochilaBin.cpp`: Problema da Mochila Binária (Knapsack Problem).
- `cutrod.cpp`: Problema do Corte de Hastes (Rod Cutting - Programação Dinâmica).
- `ssc.cpp`: Subsequência Crescente Máxima (Longest Increasing Subsequence - Programação Dinâmica).

## 🛠️ Como Executar

Para compilar e executar qualquer um dos arquivos `.cpp`, você precisará de um compilador C++ (como o `g++`).

1. **Abra o terminal** na pasta do arquivo desejado.
2. **Compile o código**:
   ```bash
   g++ nome_do_arquivo.cpp -o programa
   ```
3. **Execute o programa**:
   ```bash
   ./programa
   ```

*(Nota: No Windows, pode ser necessário usar `./programa.exe`)*
